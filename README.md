# PID_for_schools
Jednoduchý PID kontrolér a automatický ladič pro line followery založené na Arduino/raspberrypico/teensy. 

Použití
1) Do programu na lazení vložte do proměnných na začátku všechny parametry robota
2) Zapněte program
3) Zmáčkněte auto-calibrate
4) Po dokončení se slidery s Kp a Kd automaticky posunou na optimální hodnoty

1) Otevřete .ino soubor
2) Do části PHYSICAL MODEL zadejte parametry robota
3) Do části PID vložte dříve nalezené hodnoty Kp a Kd
